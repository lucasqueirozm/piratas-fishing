# 🏴‍☠️ Piratas Fishing

E-commerce de iscas e equipamentos de pesca.

## 🎣 Sobre

Piratas Fishing é uma loja virtual especializada em iscas artificiais e equipamentos de pesca. O projeto conta com:

- 🛒 Carrinho de compras interativo com seleção de quantidade
- 📱 Checkout via WhatsApp com mensagem pré-formatada
- 📄 Páginas de detalhes de produto para compras em quantidade
- 🤖 Chatbot assistente para dúvidas dos clientes
- 📐 Design responsivo e moderno

## 🛠️ Tecnologias

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)

## 🚀 Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/lucasqueirozm/piratas-fishing.git
   ```
2. Abra o arquivo `index.html` no navegador.

## 📁 Estrutura do Projeto

```
piratas-fishing/
├── assets/          # Imagens e recursos
├── index.html       # Página principal
├── product.html     # Página de detalhes do produto
├── script.js        # Lógica do carrinho e interações
├── style.css        # Estilos do projeto
└── README.md
```
